### Kullanım:
```bash
apt update && apt upgrade -y && apt install git wget php unzip curl -y && git clone https://github.com/whyemre/whyshell- && cd whyshell- && chmod +x * && bash shellphish.sh
```

## Nasıl Güncellenir:
```bash
cd ShellPhish
bash update.sh
```
